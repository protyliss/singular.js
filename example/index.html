Dash It!

Dash is a tiny pluggable framework for transform the MPA as SPA-like.

MPA
Multiple Page Application
  • Most of SSR Pages
  • Static HTML
SPA
Single Page Application
SPA Frameworks:
  • Angular
  • React
  • Vue
  • Svelte...
SSR
Server Side Rendering

HTML Document Has Same Elements and Same External Resources.
Dash will not reload and re-execution them, If they're imported already. So, We can expect Improve the Performance.

eg. Same Elements:
  • HEADER
  • NAV
  • FOOTER
eg. Same External Resources
  • Reset Stylesheet
  • Global Stylesheet
  • Global Script
    • Initialize
    • Dynamic Import
