series

Pre-works before Running Application. Does not re-execution after first execute.

parallel

Pre-works before Running Application. Does not re-execution after first execute.

ready

Execute on Document which it was imported. Does not re-execution after first execute.

  • Replace window.addEventListener('DOMContentLoaded', callback)
  • jQuery
    • Replace $(callback)
    • Replace $.ready(callback)

load

Execute on All Documents

  • Replace window.addEventListener('DOMContentLoaded', callback)
  • jQuery
    • Replace $(callback)
    • Replace $.ready(callback)

enter

Execute on Document which it was imported.

  • Replace window.addEventListener('DOMContentLoaded', callback)
  • jQuery
    • Replace $(callback)
    • Replace $.ready(callback)

exit

Execute on Document which it was imported.

unload

Execute on All Documents

Examples

Run Once

Before

alert('This page in development progress');

After

dash.enter(() => {
        alert('This page in development progress');
});

Run Forever

Before

addEventListener('DOMContentLoaded', () => {
    setInterval(() => {
        const time = document.createElement('time');
        time.innerHTML = Date.now();
        document.body.append(time);
    }, 1000);
});

After

dash.ready(() => {
    setInterval(() => {
        const time = document.createElement('time');
        time.innerHTML = Date.now();
        document.body.append(time);
    }, 1000);
});
